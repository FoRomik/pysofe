About

PySOFE is a Python software package for solving partial differential equations in 1D, 2D, and 3D using the finite element method.

Installation

PySOFE is hosted at the Python Packaging Index so you can simply install it by running

$ pip install pysofe

in a terminal. For further information please have a look at the documentation.

Documentation

The documentation can be found at readthedocs.
